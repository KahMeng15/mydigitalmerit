Excel File Format

Your Excel file should have these columns:

  • Date/Time: Event date and time
  • Name: Student full name
  • Matric Number: Student ID (S12345, DP121234, etc.)
  • Role: Student role in event
  • Additional Notes: Achievements/awards
  • Link Proof: Evidence link (optional)
